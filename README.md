# 🚗 Distracted Driver Detection using Deep Learning

This project detects **driver distraction behaviors** using CNNs trained on the **State Farm Distracted Driver Dataset**.

---

## 📂 Project Structure
```
Distracted_Driver_Detection/
│── data/ (dataset not uploaded due to size)
│── models/ (trained model)
│── notebooks/ (Colab notebook)
│── outputs/ (demo video + images)
│── docs/ (report + screenshots)
```

---

## 🚀 Usage
1. Clone Repo  
```bash
git clone https://github.com/yourusername/Distracted_Driver_Detection.git
cd Distracted_Driver_Detection
```

2. Install Requirements  
```bash
pip install -r requirements.txt
```

3. Run Training Notebook (Google Colab)  
Open `notebooks/distracted_driver_training.ipynb`

4. Run Real-Time Demo  
```bash
python demo.py
```

## 📊 Classes
- c0 : safe driving  
- c1 : texting right hand  
- c2 : talking right hand  
- c3 : texting left hand  
- c4 : talking left hand  
- c5 : adjusting radio  
- c6 : drinking  
- c7 : reaching behind  
- c8 : hair/makeup  
- c9 : talking to passenger
